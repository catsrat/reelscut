# Split-screen b-roll

When "Split-screen" is checked, the speaker goes on top and a video from this
folder fills the bottom half. Drop gameplay / satisfying / b-roll clips here
(`.mp4`, `.mov`, `.webm`). One is picked at random per run and looped to fit.

`sample_flow.mp4` is a generated placeholder (a flowing gradient) so the feature
works out of the box. **Replace it with real footage** for posting:
- Gameplay (Subway Surfers, Minecraft parkour, GTA, etc.) — the classic retention format
- "Satisfying" clips (soap cutting, hydraulic press, kinetic sand)
- Any relevant b-roll

⚠️ Use footage you have the rights to. Popular gameplay is widely used for this
format, but it can still be copyrighted — use royalty-free/your own where possible.

Tip: a 1080x960 (or wider) clip looks best; it's cropped to fill the bottom half.
